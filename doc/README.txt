Installation Instruction
1. Open videm.vlworkspace with VimLite.
2. Select 'Release_Unix' build target. ('Release_Win32' if you're on Windows)
   How to select a build target?
   Put cursor on line 1 of workspace buffer, and press <CR>.
3. Build 'CxxParser' project.
   How to build a project?
   Put cursor on project line and press '.'(dot).
4. Build 'IntExpr' project.
5. Build 'DataTypes' project.
6. Build 'vlctags2' project.
7. On Linux
     Copy 'vimlite/*' -> '~/.vimlite/'
   On Windows
     Copy 'vimlite/*' -> "$VIM/vimlite/"
